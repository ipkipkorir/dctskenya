# DCTS Kenya Website

Professional website for Diaspora Care & Trust Services Kenya - providing trusted local support for Kenyan diaspora families.

## About DCTS Kenya

DCTS Kenya offers professional, accountable diaspora care services for Kenyan families living abroad in the UK, US, Canada, Australia, and Gulf countries. We manage elderly care, property supervision, education support, government services, and more with complete transparency.

## Services Offered

1. **Elderly Care** - Hospital visits, wellness check-ins
2. **Property Management** - Land inspections, construction supervision
3. **Education Support** - School visits, academic follow-ups
4. **Government Services** - NHIF, NSSF, title deeds, documents
5. **Event Coordination** - Funeral arrangements, family events
6. **Escrow Services** - Secure fund management
7. **On-Demand Tasks** - Ad-hoc errands and urgent requests

## Pricing

- **One-Time Services**: $50 - $200 per task
- **Monthly Subscriptions**: $150 - $500/month
- **Project Supervision**: 5-10% of project value
- **Gulf Micro-Services**: $20 - $50 per task

## Target Audience

- Kenyan professionals aged 30-60
- Living in UK, US, Canada, Australia, Gulf
- Managing family/property in Kenya
- Value transparency and professionalism
